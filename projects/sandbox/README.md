# Sandbox Demo for ngx-signal-breadcrumbs

This `sandbox` project serves as a comprehensive demonstration and showcase for the `ngx-signal-breadcrumbs` Angular library. It illustrates how to integrate and use the library's features in a typical Angular application setup.

## Running Locally

To run this demo application on your local machine:

1.  **Navigate to the workspace root:**
    ```bash
    cd /home/carlesra/Projects/ANGULAR/signal-breadcrumbs-workspace
    ```
2.  **Start the development server for the sandbox project:**
    ```bash
    ng serve sandbox
    ```
3.  Open your browser and go to `http://localhost:4200/`.
