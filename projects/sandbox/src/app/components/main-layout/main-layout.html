ngx-signal-breadcrumbs

A lightweight, modern, and signal-based breadcrumb solution for Angular.

Live Preview

Installation

Install the library via npm:

npm install ngx-signal-breadcrumbs

Usage

Import and add the component to your template:


<ngx-signal-breadcrumbs />

Customize it with the available inputs:


<ngx-signal-breadcrumbs
  [showHome]="true"
  [linkColor]="'#3b82f6'"
  [linkHoverColor]="'#93c5fd'"
  [separator]="'slash'" />
